#  Module : Smart Docking System (GDIZ)

Ce module est une extension opérationnelle de mon projet de Supply Chain. Il vise à optimiser l'arrivée physique des camions dans les unités industrielles de la **Zone Industrielle de Glo-Djigbé (GDIZ)** et des cimenteries du Bénin.

##  Fonctionnalités Clés
- **Algorithme d'Assignation :** Attribution intelligente des camions aux quais disponibles pour minimiser le temps d'attente.
- **Gestion des Flux :** Réduction des embouteillages aux entrées des usines.
- **Alertes SMS (API Twilio) :** Notification automatique des chauffeurs sur leur téléphone dès que leur quai est prêt.

##  Technologies Utilisées
- **Python / Streamlit :** Interface web pour les agents de quai et managers.
- **Pandas :** Gestion des données de flux en temps réel.
- **Twilio :** Passerelle SMS pour la communication avec les transporteurs.
